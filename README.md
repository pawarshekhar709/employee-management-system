# ShopType Management System

This is a simple Django project for managing Shop Types.

## Features

- View a list of Shop Types.
- Add new Shop Types.
- Edit existing Shop Types.
- Delete Shop Types.
- Download Shop Types data in CSV and PDF formats.

## Requirements

- Python 3.x
- Django 3.x
- xhtml2pdf
- reportlab
